Folder-Capper - Managing a folder size under a certain maximum size

### Installation

```sh
$ npm install folder-capper
```


### Init

Params:

```js
{
  folderToCap
  ignores
  folderToClear
  capSize
  sortingFunction
  retryTime
  autoWatchInterval
}

```
